# Projetos da Faculdade - ADS

Repositório de projetos e estudos da faculdade de Análise e Desenvolvimento de Sistemas (ADS).

## Sobre

Este repositório contém documentação, projetos acadêmicos, recursos de estudo e anotações da jornada na formação em Análise e Desenvolvimento de Sistemas. O objetivo é organizar e compartilhar conhecimentos adquiridos durante o curso, além de servir como portfólio dos trabalhos realizados.

## Formação

Análise e Desenvolvimento de Sistemas (ADS)

## Áreas de Interesse

- Desenvolvimento Web (Frontend e Backend)
- - Programação em Python, JavaScript
  - - Administração de Sistemas Linux
    - - Computação em Nuvem
      - - Banco de Dados e Sistemas de Informação
        - - Engenharia de Software
